# YouTube Mini Lite v2

Bản không cần API key, tối ưu cho iPad Mini 1.

## Cách dùng
- Dán link YouTube để phát bằng embed nhẹ.
- Nhập từ khóa rồi mở kết quả YouTube Mobile.
- Add to Home Screen như app.

## Lưu ý
GitHub Pages không thể ép YouTube luôn 360p. Hãy chọn 144p/240p/360p trong player nếu có.
