# YouTube Mini v1

Web app nhẹ cho iPad Mini 1 / iOS 9.3.5.

## Tính năng

- Dán link YouTube để xem bằng embed nhẹ.
- Nhập từ khóa để tìm video qua YouTube Data API.
- Danh sách kết quả nhẹ, không comment, không Shorts, không trang chủ nặng.
- Add to Home Screen như app.

## Cách dùng

1. Upload `index.html` và `icon.png` lên GitHub repo.
2. Bật GitHub Pages.
3. Mở link Pages trên Safari iPad.
4. Add to Home Screen.

## Lưu ý

- Tìm kiếm cần YouTube Data API key.
- Nếu không có API key vẫn có thể dán link video để xem.
- iPad Mini 1 yếu, nên chọn chất lượng 144p/240p nếu bị lag.
