Không nhúng được trang

Một số server Invidious không cho hiển thị bên trong iframe. Bấm “Mở” để mở server hiện tại trong Safari/app web.

Cách dùng

Chọn server → nhập từ khóa như tái sinh → bấm Tìm. Nếu server lỗi, đổi server khác.

Bản này không cần API. Nó dùng Invidious như một YouTube nhẹ hơn.
