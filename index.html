YouTube Mini v1
Dán link YouTube để xem ngay, hoặc bấm API để nhập YouTube Data API key rồi tìm bằng từ khóa.

Gợi ý cho iPad Mini 1: chọn video ngắn, nếu lag hãy bấm bánh răng trong player và giảm chất lượng xuống 144p/240p.

Cài đặt

Search cần YouTube Data API key. Nếu không có key, bạn vẫn dán link YouTube vào ô tìm để xem bằng embed nhẹ. Không upload API key lên GitHub; key chỉ lưu trong Safari localStorage của iPad.
